## Strong resume bullets

```txt
- Built EventraHQ, a full-stack event operations SaaS using React, JavaScript, Node.js, Express.js, and Supabase PostgreSQL with secure REST APIs and role-based access control.

- Implemented JWT authentication, bcrypt password hashing, protected routes, and server-side authorization for user, organizer, and admin roles.

- Designed a normalized Supabase PostgreSQL schema for users, events, registrations, check-ins, and audit logs with indexed tables for scalable querying.

- Developed event marketplace features including search, category filtering, real-time seat availability, event registration, and organizer-managed event creation.

- Created an admin analytics dashboard with platform metrics including users, events, registrations, revenue, occupancy rate, check-in rate, audit logs, and category charts.

- Integrated optional Gemini/OpenAI-powered AI event brief generation to produce event strategy, agenda suggestions, operational risks, and marketing angles.

- Added production-oriented backend middleware including Helmet security headers, CORS origin control, API rate limiting, request compression, centralized error handling, and audit logging.

- Built a responsive 2026-style UI/UX with custom CSS, glassmorphism panels, premium event cards, dashboard widgets, and mobile-first layouts.
```
